# Weather iOS App

App developed to practice some iOS development concepts:
  - AutoLayout: StackViews, alignment and constraints
  - Dark mode
  - Vector assets
  - Struct
  - Optional operators
  - Protocols
  - Extensions
  - Delegates
  - Closures
  - Internal and External parameter names
  - Computed Properties
  - Delegate pattern
  - Property List
  - Core Location
  - Networking / API's
  - JSON Parsing
